# Stock-Price-Basic-Analysis
# Project Goal
The Project Goal is to analyze the histouical stock price data to understand monthly price trends.
# Dataset source
This data set came from Kaggle and the name  of the data set is Sample Sock Price Data, Author Tehmina Raja
# Steps Taken
1. Downloaded the .csv from kaggle
2. Uploaded the stock price data to BigQuery
3. Cleaned the data using SQL (by removing the missing vlaues, and convertting data types)
4. Aggregated the data to a monthly averages
5. Exported the results and created a Google Sheets
6. Saved cleaning and monthly average to files
# Findings
1. The data shows price trends that changed over time (see data, and in charts)
2. From the data it shows that some months had higher average closing prices than other months
